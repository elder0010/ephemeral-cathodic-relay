Ephemeral Cathodic Relay - CBM 8296 PET demo released at Nordlicth 2025 by Genesis Project and Distant Future. 

This demo features a new video mode exploiting the decay time of the CRT phosphors, samples and IRQ loading from the disk.
It is designed to run and make sense on real hardware only.
